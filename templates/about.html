Source link of the dataset : Twitter Data!

Definition and type of the variables:

  • TweetID - Unique identifier of a tweet - String
  • Weekday - Day of the week when the tweet was posted - String
  • Hour - Hour of the day when the tweet was posted - Integer
  • Day - Day of the month when the tweet was posted - Integer
  • Lang - Language of the tweet - String
  • isReshare - Whether the tweet is a reshare (or retweet) of another tweet- String
  • Reach - Potential audience size or reach of the tweet - Integer
  • Retweet Count - Number of times the tweet has been retweeted - Integer
  • Likes - Number of likes (or favorites) the tweet has received - Integer
  • Klout - Klout score of the user who posted the tweet - Integer
  • Sentiment - Sentiment of the tweet - Decimal
  • Text - Text content of the tweet - String
  • LocationID - Unique identifier of the location associated with the tweet. - Integer
  • UserID - Unique identifier of the user who posted the tweet - String
